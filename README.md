## Angular 4

A demo for Angular 4 using the `IdbContext` (IndexedDb) can be found 
[here](https://github.com/normalized-db/examples/tree/master/angular-demo).

Especially note [data-store.service.ts](https://github.com/normalized-db/examples/blob/master/angular-demo/src/app/core/service/data-store.service.ts) 
in `app/core/service` and [schema.ts](https://github.com/normalized-db/examples/blob/master/angular-demo/src/assets/data/schema.ts)
in `app/assets/data`.
